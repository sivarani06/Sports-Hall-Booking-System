Scheduled Jobs & Client/Server Policies

Explore the background rules and scheduled jobs automating the Sports Hall booking operations behind the scenes.

Scheduled Script Execution (Scheduled Job)

Runs Daily at 01:00 AM
SHBS Facility Auto-Maintenance Checker

Scans slots for regular weekly maintenance schedules. Sets affected slot allocations to 'Maintenance' state, preventing user bookings for those specific times.

// ServiceNow Server Script
var grSlot = new GlideRecord('x_shbs_slot');
grSlot.addQuery('maintenance_day', new GlideDateTime().getDayOfWeekUTC());
grSlot.query();
while(grSlot.next()) {
    grSlot.status = 'maintenance';
    grSlot.update();
}

Scheduled Script Execution (Scheduled Job)

Runs Every 30 Minutes
SHBS Unapproved / Idle Reservation Expirer

Finds booking requests that have been pending approval for more than 48 hours, changes their status to 'Cancelled', and notifications are dispatched to both requester and approver.

// Auto-cancels stale bookings
var grBooking = new GlideRecord('x_shbs_booking');
grBooking.addQuery('status', 'requested');
grBooking.addQuery('sys_created_on', '<', gs.daysAgo(2));
grBooking.query();
while(grBooking.next()) {
    grBooking.status = 'cancelled';
    grBooking.comments = 'System automatically closed request due to inactivity (48h limit).';
    grBooking.update();
}
